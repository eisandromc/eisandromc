# Solution, Stack Alvo Prof. Sandro Platform

## Escopo
Solução especializada para projetos que combinem backend web, tipagem forte, mobile cross-platform, automação corporativa e esteiras em GitHub Actions.

## Composição principal
- Core: investigador-de-codigo, arquiteto-de-software, implementador, testes, revisor-tecnico, documentador-tecnico
- Backend: php
- Contratos e frontend: typescript
- Mobile: flutter, dart
- Enterprise backend: csharp
- Entrega e infraestrutura: docker, aws, github-actions, devops
- Segurança e operação: seguranca, observabilidade

## Estratégia operacional
1. Investigar antes de editar.
2. Planejar antes de implementar.
3. Implementar em blocos pequenos.
4. Validar testes e contratos cruzados.
5. Revisar segurança e pipeline quando houver impacto operacional.

## Quando usar
- Produtos multi-stack com backend PHP, superfícies TypeScript, apps Flutter, serviços .NET e entrega em Docker ou AWS.
- Repositórios com forte dependência de GitHub Actions.
