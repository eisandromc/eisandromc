# Pacotes executivos de adoção

## 1. Essenciais
Pacote base recomendado para quase qualquer repositório profissional.

## 2. Enterprise
Pacote para times com mais governança, segurança, release, observabilidade e onboarding.

## 3. Web moderna
Pacote para aplicações web com frontend, backend, API, SSR e CI/CD.

## 4. Mobile
Pacote para Flutter, React Native, Android e iOS.

## 5. Cloud e DevOps
Pacote para plataforma, infraestrutura, pipelines, operação e nuvem.

## 6. Dados e IA
Pacote para ETL, APIs analíticas, notebooks, pipelines de modelagem e serviços inteligentes.

## Regra operacional
Comece por `essenciais` e some apenas o bloco do domínio principal do repositório.
