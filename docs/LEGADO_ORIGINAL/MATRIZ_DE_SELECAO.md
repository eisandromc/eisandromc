# Matriz de seleção de agentes

| Cenário | Agentes principais | Complementares |
|---|---|---|
| Entender repositório | Investigador | Onboarding |
| Nova feature | Investigador, Arquiteto, Implementador | Testes, Revisor, Documentador |
| Bug crítico | Investigador, Implementador, Testes | Revisor, Observabilidade |
| Refatoração | Investigador, Arquiteto, Refatorador | Testes, Revisor |
| API nova | Arquiteto, APIs e Integrações, Implementador | Testes, Segurança, Documentador |
| Pipeline quebrando | DevOps, GitHub Actions | Testes, Observabilidade |
| Baixa performance | Investigador, Performance | Banco de Dados, Observabilidade |
| Risco de segurança | Segurança | Revisor, DevOps |
| Projeto Flutter | Flutter, Dart | Testes, DevOps, Documentador |
| Projeto PHP Laravel | PHP, Laravel | Testes, Banco de Dados, DevOps |
| Projeto C# .NET | C#, .NET | Testes, DevOps, Observabilidade |
| Projeto TypeScript Node | TypeScript, Node.js | Testes, APIs, DevOps |
