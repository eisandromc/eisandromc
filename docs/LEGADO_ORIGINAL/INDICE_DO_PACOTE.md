## Como começar
1. Escolha o pacote mínimo ou recomendado.
2. Copie os `.agent.md` necessários para `.github/agents`.
3. Adicione skills relevantes.
4. Ajuste prompts para o padrão do seu time.
5. Valide em tarefas pequenas antes de ampliar o conjunto.
