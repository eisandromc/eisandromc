# Stack Alvo Ultra Especializado

## Escopo coberto
- PHP
- TypeScript
- Flutter
- Dart
- C#
- Docker
- AWS
- GitHub Actions

## Princípio de desenho
Menos agentes ativos por padrão, mais qualidade por handoff e skill especializada.

## Núcleo mínimo
- investigador-de-codigo
- arquiteto-de-software
- implementador
- testes
- revisor-tecnico
- documentador-tecnico

## Regras de ativação
- Ative agentes de stack apenas quando a stack existir no repositório.
- Em mudanças cross-stack, use a skill cross-stack-contract-audit.
- Em mudanças de pipeline ou cloud, acione github-actions, aws, docker e devops somente quando houver alteração operacional real.
