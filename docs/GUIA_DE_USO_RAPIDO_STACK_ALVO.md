# Guia de Uso Rápido, Stack Alvo

## Passo 1
Copie `.github/agents`, `.github/skills` e `.github/copilot-instructions.md` para o repositório alvo.

## Passo 2
Ative primeiro apenas o preset `presets/stack-alvo-ultra`.

## Passo 3
Adicione agentes de stack conforme presença real no repositório.

## Passo 4
Em PRs sensíveis, exija passagem por `testes` e `revisor-tecnico` antes de concluir.
