# Matriz de Risco e Encaminhamento

## Baixo risco
Investigador -> Arquiteto -> Implementador -> Testes -> Revisor

## Médio risco
Investigador -> Arquiteto -> Implementador -> Testes -> Segurança ou Performance -> Revisor

## Alto risco
Investigador -> Arquiteto -> Implementador -> Testes -> Segurança -> DevOps -> Revisor -> Documentador
