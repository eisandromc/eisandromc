# Matriz de seleção

## Pergunta 1
A tarefa é só leitura e diagnóstico?
- sim: investigador-de-codigo ou revisor-tecnico
- não: siga para pergunta 2

## Pergunta 2
A tarefa exige mudança arquitetural?
- sim: arquiteto-de-software
- não: siga para pergunta 3

## Pergunta 3
A tarefa exige alteração de código?
- sim: implementador ou agente de stack
- não: documentador-tecnico, onboarding ou planejador-de-tarefas

## Pergunta 4
Há risco de regressão?
- sim: testes, depois revisor-tecnico

## Pergunta 5
Há pipeline, infra ou entrega?
- sim: devops, github-actions, terraform, kubernetes, docker ou release
