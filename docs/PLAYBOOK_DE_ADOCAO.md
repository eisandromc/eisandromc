# Playbook de Adoção

## Etapa 1
Ative preset essenciais.

## Etapa 2
Adicione um preset principal de domínio, como web-moderna, mobile, cloud-devops ou dados-ia.

## Etapa 3
Ative somente os agentes de stack presentes no repositório.

## Etapa 4
Revise quais agentes são user-invocable e quais ficam apenas para uso contextual.
